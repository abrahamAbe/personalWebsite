My Creep Score

My Creep Score is a personal project I worked on, it provides Last Hit average scores per game to League of Legends players. To get these results a user has to simply type his Summoner name in the search bar, select the region where he plays and hit Go. The app displays the scores for every Champion the summoner has played.


This is a useful tool for players who are looking to improve their last hitting skills. It helps them keep track of their progress with each individual champion and every lane they've played with said champion.


I made this application using the Riot Games API as my data source, HTML, CSS, JavaScript and React for the front end, Node.js to handle REST API calls and MongoDB to store players scores.
