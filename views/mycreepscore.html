My Creep Score

My Creep Score is a personal project I worked on, it provides Last Hit and gold average scores per game to League of Legends players. To get these results a user has to simply type his Summoner name in the search bar, select the region where he plays and hit Go. The app displays the scores divided by Champions and champion roles.


This is a useful tool for players who are looking to improve their last hitting skills. It helps them keep track of their progress with each individual champion as well as champion roles. Here are two reddit posts where I got feedback from the LoL community: Post 1, Post 2.


I made this application using the Riot Games API as my data source, HTML and CSS for the front end, MySQL to manage the store/update processes for user records and Java Servlets to handle all of the API calls and data processing.
